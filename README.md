# AushadhiAI - Target Selection Application

A professional, user-friendly, and responsive web application for virtual drug screening. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Target Selection Page**: Users can enter a disease name and initiate the virtual screening process.
- **Interactive Loading Animation**: Displays a visually appealing animation while the screening process is running.
- **Results Display**: Shows identified potential drug targets with confidence scores and associated pathways.

## Technology Stack

- **Next.js**: React framework with server-side rendering for better performance and SEO.
- **TypeScript**: For type safety and better developer experience.
- **Tailwind CSS**: Utility-first CSS framework for responsive design.
- **Framer Motion**: For smooth animations and transitions.
- **Axios**: For making API requests.
- **React Icons**: For high-quality icons.
